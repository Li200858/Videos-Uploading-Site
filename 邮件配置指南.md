# 邮件配置快速指南

## 快速配置步骤

### 方式一：使用163邮箱（推荐，简单）

1. **获取163邮箱授权码**
   - 登录163邮箱
   - 进入"设置" → "POP3/SMTP/IMAP"
   - 开启"POP3/SMTP服务"
   - 生成"授权码"（不是登录密码！）

2. **配置环境变量**
   
   在 `.env.local` 文件中添加：
   ```bash
   SMTP_HOST="smtp.163.com"
   SMTP_PORT="465"
   SMTP_USER="你的163邮箱@163.com"
   SMTP_PASSWORD="你的授权码"
   SMTP_FROM="noreply@yourdomain.com"
   ```

### 方式二：使用QQ邮箱

1. **获取QQ邮箱授权码**
   - 登录QQ邮箱
   - 进入"设置" → "账户"
   - 开启"POP3/SMTP服务"
   - 生成"授权码"

2. **配置环境变量**
   ```bash
   SMTP_HOST="smtp.qq.com"
   SMTP_PORT="587"
   SMTP_USER="你的QQ邮箱@qq.com"
   SMTP_PASSWORD="你的授权码"
   SMTP_FROM="noreply@yourdomain.com"
   ```

### 方式三：使用Gmail

1. **获取Gmail应用专用密码**
   - 登录Google账号
   - 进入"安全性" → "两步验证"
   - 生成"应用专用密码"

2. **配置环境变量**
   ```bash
   SMTP_HOST="smtp.gmail.com"
   SMTP_PORT="587"
   SMTP_USER="你的Gmail@gmail.com"
   SMTP_PASSWORD="你的应用专用密码"
   SMTP_FROM="noreply@yourdomain.com"
   ```

## 在Render中配置

1. 登录 Render Dashboard
2. 选择您的服务
3. 进入 "Environment" 标签
4. 添加以下环境变量：
   - `SMTP_HOST`
   - `SMTP_PORT`
   - `SMTP_USER`
   - `SMTP_PASSWORD`
   - `SMTP_FROM`（可选）

## 测试配置

配置完成后，创建测试邀请，检查学生邮箱是否收到邮件。

## 注意事项

- **163/QQ邮箱**：必须使用"授权码"，不是登录密码
- **Gmail**：必须使用"应用专用密码"，不是普通密码
- 如果未配置邮件服务，邀请功能仍然可用，但需要手动复制链接发送
